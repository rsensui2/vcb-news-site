# vcb-news-site

Auto-generated output repository for the [Vibe Coder Bootcamp](https://vibe-coder-bootcamp.com) news hub.

- Built and deployed daily by the `ryoko` news pipeline (`scripts/build-news-pages.js` + `scripts/deploy-news-site.sh`).
- **This repo is output-only.** Do not edit files here by hand — they will be overwritten on the next cron run.
- Source: `~/ryoko/docs/news-site/` (symlinked with `~/.openclaw/workspace/docs/news-site/`).

Live: https://rsensui2.github.io/vcb-news-site/
