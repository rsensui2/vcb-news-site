概要

Claude CodeやAntigravityの能力を拡張する「Skill」のまとめリスト。投稿者が「脳死でこのリストから入れておけば間違いない」と推奨している。

投稿者おすすめのスキル

| スキル名 | 説明 |

|---------|------|

| deep-research | 市場調査・競合分析を自律実行 |

| revealjs-skill | プレゼン資料を爆速生成 |

| test-driven-development | TDDで実装をガイド |

| brainstorming | AIと壁打ちしてアイデアを洗練 |

| ship-learn-next | 次に何を学ぶか、作るべきかを提案 |

| video-downloader | 動画教材などを保存 |

| invoice-organizer | 請求書などの整理を自動化 |

全スキルリスト（カテゴリ別）

📄 ドキュメント系

  • docx - Word文書の作成・編集・分析
  • pdf - PDF抽出、テーブル、メタデータ、マージ、注釈
  • pptx - スライドの読み込み・生成・調整
  • xlsx - スプレッドシート操作（数式、チャート、データ変換）
  • revealjs-skill - Reveal.jsでプロ品質のプレゼン生成

🛠 開発・コードツール

  • web-artifacts-builder - React, Tailwind, shadcn/uiでHTMLアーティファクト作成
  • test-driven-development - TDD実装ガイド
  • using-git-worktrees - 隔離されたgit worktree作成
  • finishing-a-development-branch - 開発ブランチ完了ワークフロー
  • pypict-claude-skill - ペアワイズテスト設計
  • aws-skills - AWS CDKベストプラクティス
  • claude-starter - 40個のスキル付きテンプレート
  • move-code-quality-skill - Move言語品質チェック
  • claude-code-terminal-title - ターミナルタイトル管理
  • charles-proxy-extract - Charles ProxyセッションHTTPトラフィック分析
  • plugin-authoring - Claude Codeプラグイン作成ガイド
  • claude-skills - 65個のフルスタック開発スキル
  • jules - Google Jules AIエージェント連携

📊 データ・分析

  • root-cause-tracing - エラーの根本原因追跡
  • csv-data-summarizer - CSV自動分析（列、分布、欠損、相関）
  • postgres - 安全なPostgreSQL読み取りクエリ

🔬 科学・リサーチ

  • claude-scientific-skills - 125+の科学スキル
  • materials-simulation-skills - 計算材料科学シミュレーション
  • deep-research - Gemini Deep Researchエージェントで自律リサーチ

✍️ ライティング・リサーチ

  • article-extractor - Webページから記事テキスト抽出
  • content-research-writer - リサーチ・引用・フィードバック付きコンテンツ作成
  • internal-comms - 社内コミュニケーション文書作成
  • brainstorming - アイデアを構造化質問で洗練
  • family-history-research - 家系図・系譜リサーチ支援

📘 学習・ナレッジ

  • tapestry - ドキュメントを知識ネットワークに相互リンク
  • ship-learn-next - 次に何を作る/学ぶかの提案

🎬 メディア・コンテンツ

  • youtube-transcript - YouTube動画の文字起こし取得・要約
  • video-downloader - 動画ダウンロード
  • image-enhancer - 画像品質向上（スクリーンショット特化）
  • imagen - Google Gemini画像生成
  • claude-epub-skill - EPUBパース・分析
  • video-prompting-skill - 動画生成モデル用プロンプト作成

🏥 ヘルス・ライフサイエンス

  • claude-ally-health - 医療レポート分析、健康メトリクス追跡

🤝 コラボ・プロジェクト管理

  • git-pushing - git操作自動化
  • linear-claude-skill - Linear連携（イシュー、プロジェクト、チーム管理）
  • linear-cli-skill - Linear CLI操作
  • meeting-insights-analyzer - ミーティング文字起こし分析
  • review-implementing - コード実装レビュー
  • test-fixing - 失敗テスト検出・修正提案
  • plannotator - 計画レビューUI
  • outline - Outlineウィキ連携
  • google-workspace-skills - Gmail, カレンダー, Chat, Docs, Sheets, Slides, Drive連携

🛡 セキュリティ・Webテスト

  • defense-in-depth - 多層テスト・セキュリティベストプラクティス
  • ffuf_claude_skill - FFUFファジング連携
  • systematic-debugging - 体系的デバッグ
  • Trail of Bits Security Skills - 静的解析、コード監査
  • varlock-claude-skill - 環境変数セキュア管理
  • webapp-testing - Playwrightでローカルアプリテスト

🔧 ユーティリティ・自動化

  • file-organizer - ファイル・フォルダ整理
  • invoice-organizer - 請求書・領収書自動整理
  • skill-creator - 新規スキル作成テンプレート
  • template-skill - 最小スケルトン構造
