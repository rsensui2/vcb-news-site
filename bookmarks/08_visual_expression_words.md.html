ビジュアル表現を引き出す言葉

概要

「あのデザインをAIで再現したいんだけど、結局なんて言えばいいんだ？」という問題を解決する語彙集。デザインの歴史を遡りながら、AIから望むビジュアル表現を引き出すための「言葉」を整理。

デザインスタイルの歴史的分類

1. 黎明期〜モダニズム（19世紀末〜1910年代）

| スタイル | 年代 | 特徴 |

|---------|------|------|

| アーツ・アンド・クラフツ | 1880年代〜1910年代 | 中世の手仕事への回帰、クラフトマンシップ |

| アール・ヌーヴォー | 19世紀末〜20世紀初頭 | 植物などの有機的な曲線 |

| アール・デコ | 1910年代〜1930年代 | 幾何学的で都会的、豪華さと力強さ |

| 未来派 (Futurism) | 1909年〜1940年代 | 速度・機械・ダイナミズム賛美 |

| ロシア構成主義 | 1910年代〜1920年代 | 赤・黒・白の配色、ダイナミックな幾何学 |

| デ・ステイル | 1917年〜1930年代 | 水平・垂直線と三原色のみ（モンドリアン） |

| バウハウス | 1919年〜1933年 | 「形は機能に従う」機能主義 |

2. ミッドセンチュリーと機能美（1940年代〜1950年代）

| スタイル | 特徴 |

|---------|------|

| ミッドセンチュリー・モダン | ポップで親しみやすい、鮮やかな色や不揃いな形 |

| ポップアート | 大衆文化をアートに、鮮やかな色彩と反復 |

| ブルータリズム | 素材の質感、構造の力強さ、荒々しい美学 |

| スイス・スタイル | グリッドシステムとサンセリフ体、機能的デザイン |

3. カウンターカルチャーと多様化（1960年代〜1980年代）

| スタイル | 特徴 |

|---------|------|

| サイケデリック | 極彩色と歪んだ文字、幻覚的 |

| ニュー・ウェーブ / スイス・パンク | グリッドをあえて崩した実験的スタイル |

| ポストモダニズム | 過去の様式の引用、装飾の復活、ユーモア |

| パンク | DIY精神、切り貼り文字、コピーの汚れ |

| メンフィス | パステルカラーと幾何学パターン、遊び心 |

| サイバーパンク | ネオンカラー、グリッチ、ハイテクと退廃 |

| ピクセルアート | デジタルな懐かしさ |

4. デジタル黎明期（1990年代〜2000年代）

| スタイル | 特徴 |

|---------|------|

| グランジ | 汚れたテクスチャ、破壊的スタイル |

| スキューモーフィズム | 現実世界の質感（革、木目、金属）を模倣 |

| アシッドグラフィックス | 毒々しいネオンカラー、クロームの質感 |

| Y2K | サイバーでポップ、メタリック、初期ネットの雰囲気 |

| フルティガー・エアロ | 光沢、水滴、草原、ハイテクでクリーンな未来 |

5. スマホ時代のスタンダード（2010年代）

| スタイル | 特徴 |

|---------|------|

| ヴェイパーウェイヴ | 80-90年代の消費文化をノスタルジックにコラージュ |

| フラットデザイン | 立体感排除、色面とアイコン |

| マテリアルデザイン | フラット+物理法則、影と意味のある動き |

| コーポレート・メンフィス | フラットな塗り、極端に大きな手足、非現実的な肌色 |

| ソーラーパンク | 自然とテクノロジーの共生、希望に満ちた未来 |

6. ポスト・フラットと現在（2020年代〜）

| スタイル | 特徴 |

|---------|------|

| アンチデザイン | 読みにくい色、崩れたレイアウト、違和感を与える |

| グラスモーフィズム | すりガラスのぼかし効果、半透明レイヤー |

| ネオブルータリズム | 原色、太い線、デフォルトフォントの現代的再解釈 |

| クレイモーフィズム | 粘土のようなマットな質感、丸みを帯びた3D |

| ベントーグリッド | 幕の内弁当のような四角いグリッドレイアウト |

| ニューナチュラリズム | アースカラー、手書き文字、オーガニック |

| マキシマリズム | 「More is More」空間を埋め尽くす |

感情・空気感を表現する言葉

幻想的・非現実的

| 言葉 | 意味 | デザイン表現 |

|------|------|-------------|

| Ethereal（エセリアル） | 天上的な美しさ、浮遊感 | グロー効果、柔らかなグラデーション、淡い白やパステル |

| Whimsical（ウィムジカル） | 奇妙な、遊び心のある | 不規則なカーブ、手書き風イラスト、非現実的なサイズ対比 |

| Celestial（セレスティアル） | 天上の、神々しい、宇宙的 | 星座、月、太陽、金箔、濃紺や深い紫 |

| Dreamy（ドリーミー） | 夢見心地な | パステルグラデーション、ソフトフォーカス、雲、霞 |

情緒的・静寂・暗さ

| 言葉 | 意味 | デザイン表現 |

|------|------|-------------|

| Melancholic（メランコリック） | 静かな哀愁、心地よい孤独感 | 彩度を抑えた色、セピア調、粒子感、深い影 |

| Moody（ムーディ） | 憂鬱な、情緒的な | 暗いトーン、ドラマチックな照明 |

| Nostalgic（ノスタルジック） | 郷愁、懐かしい | セピア色、色あせたフィルム質感、ノイズ |

| Serene（セリーン） | 穏やかな、平穏な | 余白の多用、淡いアースカラー、自然光 |

| Ephemeral（エフェメラル） | 儚い、つかの間の | 細いフォント、半透明オブジェクト、繊細な線画 |

活発・混沌・強さ

| 言葉 | 意味 | デザイン表現 |

|------|------|-------------|

| Vibrant（ヴァイブラント） | 活気に満ちた、鮮烈な | 彩度の高いビビッドカラー、高コントラスト |

| Eclectic（エクレクティック） | 折衷的な、多様な | 異なる時代やスタイルをあえて混在 |

| Kitsch（キッチュ） | 俗悪だが愛らしい | 原色のプラスチック感、過剰なデコレーション、ダサかわいい |

| Bold（ボールド） | 大胆な、力強い | 極太タイポグラフィ、巨大な要素配置、強い黒 |

洗練・質感

| 言葉 | 意味 | デザイン表現 |

|------|------|-------------|

| Sophisticated（ソフィスティケイテッド） | 洗練された、都会的な | 細めのセリフ体、ゴールドやシルバー、モノトーン |

| Rustic（ラスティック） | 素朴な、田舎風の | 木目、紙のざらつき、手書き文字、茶系や緑 |

| Gritty（グリッティ） | ざらざらした、気骨のある | コンクリート、アスファルト、スプレーアート |

| Sleek（スリーク） | なめらかな、つやのある | 流線型、光沢のある黒や銀、継ぎ目のない表面 |

まとめ
