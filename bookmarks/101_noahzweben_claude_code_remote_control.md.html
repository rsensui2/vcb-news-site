Claude Code 新機能, Remote Control で外出先からセッション継続

№109 / 308 GHGitHub ↗
109BOOKMARK / 308

有益度: ⭐⭐⭐⭐⭐

投稿日時: 2026-02-25 (JST)

元URL: https://x.com/noahzweben/status/2026371260805271615

要約

Claude Code に新機能 *Remote Control* が追加され、ローカルで始めたセッションをスマホから継続操作できるようになった。外出や移動中でも作業の流れを止めずに続けられるのが大きな価値。

本文

Claude Code の新機能 *Remote Control* を発表します。現在、Max ユーザー向けにリサーチプレビューとして順次展開中です。/remote-control で試せます。

ターミナルでローカルセッションを開始して、その後スマホから続きを操作できます。

散歩に出たり、日光を浴びたり、犬の散歩をしたりしても、作業の流れを失わずに済みます。

要約（日本語）

*Claude Code* に新機能 *Remote Control* が登場。Maxユーザー向けのリサーチプレビューとして展開中。

  • /remote-control コマンドで起動
  • ターミナルでローカルセッションを開始し、その後スマホで続きを操作
  • 散歩中や外出中でも作業フローを維持しやすい

開発中のセッションをモバイルから扱える、かなり実用的な機能。
