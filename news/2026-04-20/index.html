Claude Design 公開 — ブランドデザインシステムを自動取込

Anthropic Labs が『Claude Design』をリサーチプレビューとして公開。Claude との会話だけでプロトタイプ・スライド・1枚資料を生成でき、コードベースとデザインファイルを読み込んでブランドデザインシステムを自動構築・適用する機能まで備える。

キーポイント

  • 駆動モデルは Claude Opus 4.7（最新ビジョンモデル)
  • 入力: テキスト/画像/DOCX/PPTX/XLSX/コードベース/Webキャプチャ
  • 出力: Canva 連携 / PDF / PPTX / Claude Code へのハンドオフ
  • インラインコメント・直接編集・カスタムスライダーで反復改善
  • Pro / Max / Team / Enterprise で順次ロールアウト
スライド・LP制作の起点が『Figma → Claude Design → Code』に統合される可能性。VBC受講者の制作パイプライン候補として一次資料化が必須。
