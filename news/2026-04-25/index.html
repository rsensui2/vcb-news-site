Matt Shumer の GPT-5.5 詳細レビュー — 「日常では実感しにくいが押し込むと別物」

Matt Shumer が GPT-5.5 の長文レビューを公開。結論は「明確なアップグレードだが、従来モデルが既に異常な水準だったため日常 eng では実感しにくい」「真の差はデザイン・ネイティブ/モバイル・セキュリティ・低 reasoning モード・ハーネスで現れる」「Pro は速いが書き味と判断にレグレッション」というニュアンス重視の評価。

キーポイント

  • セキュリティレビュー用途で Opus / 過去 GPT が見逃した実脆弱性を検出 — 汎用モデル最強クラス
  • GPT Image 2 のモック → GPT-5.5 実装の組み合わせが Opus 並みの UI 出力に到達
  • Swift / Mac ネイティブ・モバイル開発が一段と安定
  • 低〜中 reasoning モードでも実用品質、Opus 化した手触り（速くて楽しい）
  • Pro は 1文1改行のクセや判断ミスがあり執筆用途は推奨しない
  • 単体モデル性能は飽和、マルチエージェント・ハーネスが次の伸びしろ
GPT-5.5 を Codex 置き換えだけで判断するのは過小評価。セキュリティ監査・既存 UI 拡張・ネイティブアプリ開発で明確な優位があり、VBC / TEKION でもコードレビュー工程に GPT-5.5 を組み込む実務価値が高い。
